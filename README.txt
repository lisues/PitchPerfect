###PitchPerfect###


========================================================================
DESCRIPTION:
An iPhone app that will allow users to record their voice and will then modulate the recorded audio to sound like a Chipmunk or Darth Vader. This app will also let the user speed up or slow down the rate of playback, and experience fun echo and reverb effects.


Supported Formats


=========================================================================
BUILDS REQUIREMENTS:


Xcode 8.2.1,  IOS SDK 8.2 or better, 


=========================================================================
RUNTIME REQUIREMENTS:


iPhone, iPad, or iPod Touch running iOS 8.2.1 or better Xcode 8.2.1,  IOS SDK 8.2.1 or better, 


=========================================================================
PACKAGING LIST:


RecordSoundsViewController.swift
A UIViewController provides a root view controller to be the start point of the app and provides user interface and functionalities for voice recording. 


PlaySoundsViewController.swift
A UIViewController provides provides the user optionals for playing the recorded sound.


PlaySoundsViewController+Audio.swift
Extension of PlaySoundsViewController to provide general optionals for playing sounds.
